## Project ThingPle

ThingPle은 사람의 모든것이라는 의미로 소상공인 및 자영업자들을 위한 서비스 입니다.  
라즈베리파이를 활용한 COVID-19로 인한 5인 이상 모임 금지, 9인 이상 모임 금지 등에 활용이 가능합니다.  
하루 총 방문한 인원 수 등을 추척하면 하루 매출 및 마케팅으로도 활용이 가능 합니다.
